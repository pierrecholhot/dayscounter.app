# Days Counter

Simple date calculator to track and count days since and days until a certain date

## First run

```
git clone https://github.com/pierrecholhot/dayscounter.app/
cd dayscounter.app
npm install
```

## Develop

```
npm start
```

## Deploy

```
npm run build
```
